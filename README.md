# NUTZ-ONEKEY
NUTZ一键脚手架

仓库
```
<repositories>
		<repository>
			<id>dgj nexus</id>
			<name>Team Nexus Repository</name>
			<url>http:/http://maven.kerbores.com:11981/nexus//content/groups/public</url>
		</repository>
	</repositories>
```

构建命令
```
mvn eclipse:eclipse -Dwtpversion=1.0
```
启动应用
